# Specs

Esta pasta contém as specs versionadas do Copia e Cola.

Cada spec deve ser pequena, verificável e orientada a um incremento fechado do MVP.

## Convenção

- `001-*`: base do projeto e scaffold;
- `002-*`: contexto e domínio;
- `003-*`: persistência local;
- `004-*`: CRUD de itens;
- `005-*`: experiência do popup;
- `006-*`: portabilidade dos dados;
- `007-*`: empacotamento e validação.
- `008-*`: fonte visual do Claude Design;
- `009-*`: refinamento pixel-fiel do popup;
- `010-*`: páginas internas da extensão;
- `011-*`: materiais da Chrome Web Store;
- `012-*`: landing page pública;
- `013-*`: validação visual e prontidão de publicação.
- `014-*`: migração para React/WXT e paridade visual sustentável.

## Status

- `Planned`: ainda não implementada;
- `In progress`: implementação iniciada;
- `Done`: implementada e validada;
- `Deferred`: escopo adiado.

## Fonte visual atual

Para as specs `008` a `013`, a referência canônica versionada é `design-system/screens/telas-da-extensao.html`.

O pacote local original `Extensão copia e cola.zip` fica fora do Git. O resumo histórico do ZIP está arquivado em `docs/archive/legacy-zip-asset-map.md` e não deve ser usado como plano de implementação atual.
