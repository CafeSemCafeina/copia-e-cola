# Specs

Esta pasta contém as specs versionadas do Copia e Cola.

Cada spec deve ser pequena, verificável e orientada a um incremento fechado do MVP.

## Convenção

- `001-*`: base do projeto e scaffold;
- `002-*`: contexto e domínio;
- `003-*`: persistência local;
- `004-*`: CRUD de itens;
- `005-*`: experiência do popup;
- `006-*`: portabilidade dos dados;
- `007-*`: empacotamento e validação.

## Status

- `Planned`: ainda não implementada;
- `In progress`: implementação iniciada;
- `Done`: implementada e validada;
- `Deferred`: escopo adiado.
