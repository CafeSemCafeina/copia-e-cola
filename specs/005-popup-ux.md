# Spec 005 - UX do popup

Status: Planned

## Objetivo

Construir uma experiência de popup simples, rápida e adequada ao uso operacional diário.

## Escopo

- Cabeçalho com nome do produto e domínio atual.
- Campo de título.
- Campo de conteúdo.
- Ação principal `Salvar`.
- Lista de itens do domínio atual.
- Seção de itens globais, se existirem.
- Busca simples por título e conteúdo.
- Estados vazio, erro e sucesso.
- Layout compacto para uso rápido.

## Fora de escopo

- Página de marketing.
- Dashboard completo.
- Design system complexo.
- Animações pesadas.

## Critérios de aceite

- O popup é utilizável em largura típica de extensão.
- Textos longos não quebram o layout.
- É possível salvar e copiar sem rolagem excessiva em listas pequenas.
- A busca filtra itens localmente.
- O estado vazio orienta sem excesso de texto.

## Direção de linguagem

Usar português direto e operacional:

- `Salvar`;
- `Copiar`;
- `Editar`;
- `Excluir`;
- `Itens deste site`;
- `Itens globais`;
- `Nenhum item salvo para este site ainda`.
