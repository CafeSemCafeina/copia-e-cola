Política de Privacidade

Última atualização: 20 de junho de 2026.

Resumo

O Copia e Cola é uma extensão local-first. Ela não exige conta, não usa analytics de terceiros, não sincroniza dados em nuvem e não envia seus textos para servidores.

Dados armazenados

A extensão salva localmente os textos, títulos, domínios, marcações de fixado e datas operacionais necessárias para listar, buscar, editar, copiar, exportar e importar seus itens.

Permissões do navegador

A permissão activeTab é usada para identificar o domínio da aba atual quando você abre o popup. A permissão storage é usada para guardar os itens no armazenamento local do navegador.

Compartilhamento

O Copia e Cola não vende, compartilha nem transfere seus textos para terceiros. Exportações e importações acontecem por arquivo JSON escolhido por você.

Remoção de dados

Você pode excluir itens pela própria extensão. Também pode remover todos os dados desinstalando a extensão ou limpando os dados de extensão no navegador.

Contato e código

O código-fonte está disponível em github.com/CafeSemCafeina/copia-e-cola. Use o repositório para revisar o funcionamento ou abrir uma issue pública.
