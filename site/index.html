Copia e Cola

Salve textos que você repete e cole de volta com um clique, organizados pelo site em que você está.

Instalar localmente Ver código no GitHub

Sem conta, sem nuvem e sem promessa de IA: primeiro, resolver o copiar e colar diário.

Como funciona

Três passos para reutilizar textos sem misturar contextos

1

Abra um site

WhatsApp Web, portal jurídico, CRM ou ferramenta interna viram contextos separados.

2

Salve em segundos

Cole o texto, dê um título opcional e escolha entre item do site ou item global.

3

Copie com 1 clique

Busque, fixe e copie respostas, protocolos e modelos curtos quando precisar.

Privacidade

Seus dados ficam no navegador

A extensão usa apenas activeTab para identificar o domínio da aba atual e storage para salvar seus itens localmente. Backup e importação acontecem por JSON escolhido por você.

Instalação

Publicação na Chrome Web Store em preparação

Enquanto a ficha pública não estiver no ar, gere o build e carregue a pasta dist/ como extensão sem compactação no Chrome ou Edge.

  1. Rode npm install e npm run build.
  2. Abra chrome://extensions.
  3. Ative o modo desenvolvedor.
  4. Clique em “Carregar sem compactação”.
  5. Selecione a pasta dist/ deste repositório.

FAQ

O Copia e Cola sincroniza entre computadores?

Não. O MVP é local-first. Para levar dados para outro perfil, use exportação e importação em JSON.

A extensão envia meus textos para servidores?

Não. Os itens ficam no storage local do navegador.

Existe IA ou sugestão automática?

Não nesta versão. A prioridade é um fluxo simples, rápido e auditável de salvar e reutilizar textos.
